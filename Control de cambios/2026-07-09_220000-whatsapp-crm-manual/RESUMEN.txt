Cambio: WhatsApp CRM manual para PreventWork
Fecha: 2026-07-09

Objetivo
- Crear un primer modulo sin costos adicionales para gestionar prospectos que llegan por WhatsApp Business gratis.
- Mantener el flujo manual, con apoyo de ChatGPT, sin usar WhatsApp Cloud API, n8n ni Copilot Studio.
- Centralizar este flujo en el Super Administrador, separado de los roles cliente.

Archivos principales
- admin/whatsapp_crm.php
- config/whatsapp_crm_schema.php
- database/migrations/2026_07_09_whatsapp_crm_prospectos.sql
- components/sidebar.php
- marketing-publicidad/whatsapp-crm/README.md
- marketing-publicidad/whatsapp-crm/flujo-comercial.md
- marketing-publicidad/whatsapp-crm/prompts-chatgpt.md
- marketing-publicidad/whatsapp-crm/mensajes-whatsapp-business.txt

Implementacion
- Se agrego una tabla nueva: whatsapp_prospectos.
- Se agrego una pantalla de super admin: admin/whatsapp_crm.php.
- Se agrego acceso en el menu lateral del super admin: WhatsApp CRM.
- El modulo permite crear, editar, eliminar, filtrar y dar seguimiento a prospectos.
- Cada prospecto puede abrir WhatsApp con mensaje prellenado.
- Cada prospecto puede copiar un prompt para pegar en ChatGPT.
- Se agrego documentacion comercial y mensajes rapidos sugeridos para WhatsApp Business.

Notas
- Este modulo no automatiza respuestas ni lee chats de WhatsApp.
- Para bot automatico oficial se necesitara WhatsApp Business Platform / Cloud API.
- El alcance actual esta pensado para trabajar con WhatsApp Business gratis y registro manual.
- Los roles representante, sst y trabajador son cuentas cliente; no deben controlar este CRM comercial global.
- Quedo un prospecto de prueba visible para revision: Vertix / Arturo 00.
- Se realizaron mensajes de prueba autorizados por el usuario en el chat Arturo 00 para validar el flujo manual.
